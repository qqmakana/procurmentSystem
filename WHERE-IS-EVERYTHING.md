# 📍 Where Is Everything? Quick Reference

## 🎯 "I can't find the Add Item button!"

### ✅ The Add Item button IS there!

**Location**: Details Tab → Line Items Section

**How to see it:**
1. Login first (very important!)
2. Click the **"📋 Details"** tab (first tab)
3. Scroll down to the **"Line Items"** section
4. You'll see a big green button: **"➕ Add Item"**

**If you still don't see it:**
- Make sure you're logged in (see login section at top)
- Try refreshing the page (Ctrl+R or Cmd+R)
- Check if you're on the right tab (Details, not Documents or Submit)

---

## 📂 "Where do I upload documents?"

### ✅ Documents Tab

**Location**: Second tab from left

**Look for**: **"📁 Documents"** tab

**What you'll see:**
1. **Big blue banner** at top: "How Document Upload & Approval Works"
2. **Upload area** with instructions
3. **List of uploaded documents** below

**How to upload:**
1. Login
2. Click **"📁 Documents"** tab
3. Drag files into the upload area OR click to browse
4. Files appear in the list below

---

## 📤 "How do I submit for approval?"

### ✅ Submit Tab

**Location**: Third tab (after Details and Documents)

**Look for**: **"📤 Submit"** tab

**What you'll see:**
1. **Submission Status** banner at top
2. **Requirements checklist** (what's complete)
3. **Approval workflow preview** (who will approve)
4. **Submit button** at bottom

**Steps:**
1. Complete Details tab (add line items)
2. Upload documents (Documents tab)
3. Go to Submit tab
4. Click **"Submit for Approval"** button

---

## 👁️ "Where can approvers see documents?"

### ✅ Queue Tab (for approvers)

**Location**: Fourth tab

**Look for**: **"📋 Queue"** tab

**What approvers see:**
1. List of pending requisitions
2. For each requisition:
   - **"▶ View Details"** button on the right
   - Click it to expand
   - **"📎 Supporting Documents"** section appears
   - Documents listed with **Download** buttons

**How approvers review:**
1. Login as approver
2. Click **"📋 Queue"** tab
3. Find requisition to review
4. Click **"▶ View Details"**
5. See documents, line items, everything
6. Click **"✅ Review & Approve"** to approve/reject

## 🚨 Common Issues

### "I don't see any tabs"
- **Solution**: You need to login first! Look at top of page for login form.

### "Add Item button is missing"
- **Solution**: 
  1. Make sure you're logged in
  2. Click "Details" tab (first tab)
  3. Scroll down to Line Items section
  4. The green "➕ Add Item" button should be there

### "Can't upload documents"
- **Solution**:
  1. Login first
  2. Go to "Documents" tab (second tab)
  3. Make sure file is under 10MB
  4. Make sure file type is supported (PDF, Word, Excel, Images)

### "Approvers can't see my documents"
- **Solution**:
  1. Make sure you uploaded documents in "Documents" tab
  2. Make sure you clicked "Submit for Approval" in "Submit" tab
  3. Approvers need to expand details by clicking "▶ View Details"
  4. Documents section will appear after expansion

### "Docker not working"
- **Solution**:
  1. Make sure Docker Desktop is running
  2. Run: `docker-compose down`
  3. Run: `npm run build` (build first!)
  4. Run: `docker-compose up --build -d`
  5. Open: http://localhost:8080

### "Changes not showing"
- **Solution**:
  1. Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  2. Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
  3. Close and reopen browser
  4. Make sure you rebuilt: `npm run build`
